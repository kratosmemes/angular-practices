Manipulando con @ViewChild y @ViewChildren

Usando @ViewChild para acceder a elementos del DOM en Angular. No recomendado para manipulación directa del DOM, pero útil para ciertos casos. Puede se inseguro en aplicaciones con renderizado del lado del servidor. Alternativas incluyen Renderer2 y ElementRef

This is a div element.
@for (item of items; track $index) {
Item {{item}}
}

Manupulando con Renderer2 y ElementRef

Usando Renderer2 para manipular, es mas seguro ya que no actualiza directamente el DOM, sino que usa una abstracción para hacerlo, como una copia, un espejo.

Contenido
@for (item of items2; track $index) {
Item {{item}}
}
